<h2>일러스트레이터 기본 단축키</h2>
<hr>
1. 도형 그리기<br>
 -사이즈 없이 임의적으로 드래그<br>
 -스테이즈에 클릭후 사이즈 입력<br>
 -alt+드래그(중심점이 기준)<br>
 -shift+드래그(가로 세로 비율 맞춤)<br>
 -alt+shift+드래그(중심점을 기준으로 
  가로세로 비율 맞춤)<br>
1-1. 크기 조절<br>
 - 임의적으로 크기 조절<br>
 - alt+드래그<br>
 - shift+드래그<br>
 - alt+shift+드래그<br>
2. 선택툴로 개체 선택<br>
 -shift+클릭<br>
 -도형에 조금이라도 걸처도 선택<br>
 -그룹 설정<br>
3. 복사<br>
 -alt+드래그<br>
 -alt+shift+드래그 <br>
4.정돈<br>
 - shift+ctrl+] --> 맨앞으로 가져오기<br>
 - ctrl+] --> 앞으로 가져오기<br>
 - ctrl+[ --> 뒤로 보내기<br>
 - shift+ctrl+[ --> 맨 뒤로 보내기<br>
